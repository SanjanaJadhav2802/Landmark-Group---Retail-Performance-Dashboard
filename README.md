# 🛍️ Landmark Group Retail Performance Dashboard - Power BI Project

## 📌 Overview
This project presents a comprehensive **Retail Performance Dashboard** built using Power BI for analyzing **Landmark Group sales data**.

The dashboard is designed to provide **end-to-end business insights** across sales, products, customers, and store performance.  
It enables stakeholders to track KPIs, identify trends, and make data-driven decisions.

---

## 🎯 Objectives
- Monitor overall retail performance
- Analyze sales trends over time
- Evaluate product and brand performance
- Understand customer behavior
- Assess store-level performance
- Enable deep-dive analysis using drill-through

## 📈 Key Features & Analysis

### 📊 Executive Overview
- High-level KPIs for quick business understanding  
- Sales trends over time (monthly/yearly)  
- Brand-wise sales performance  

---

### 🏷️ Sales & Product Insight
- Sales distribution across product categories  
- Brand-level performance comparison  
- Identification of top-performing products  
- Contribution analysis by category  

---

### 👥 Customer & Store Insight
- Customer purchasing behavior  
- Store-wise performance analysis  
- Geographic distribution of sales  
- Customer segmentation insights  

---

### 🔍 Detailed Sales Analysis (Drill-through)
- Drill-through functionality for in-depth exploration  
- Transaction-level or granular sales insights  
- Enables root-cause analysis of trends
